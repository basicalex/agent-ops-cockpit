# Composition Catalog

Human/agent navigation layer for HyperFrames source compositions. HyperFrames may list every HTML file; AOC uses paths, this catalog, and `index.html` as the active workbench.

## Operating rules

- `index.html` points at the current active composition under review.
- `_playgrounds/` contains boards, experiments, and internal review surfaces.
- `components/` contains reusable motion primitives that remain previewable.
- `ads/`, `social/`, and `landing/` compositions need matching shotlists.
- Update this catalog when adding, moving, or retiring compositions.
