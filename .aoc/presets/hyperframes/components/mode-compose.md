# Compose mode

Create and iterate HyperFrames videos within the AOC asset/campaign system.

Use the `aoc-hyperframes` skill to confirm whether work belongs in reusable components, brand/system compositions, or a campaign pack. Use the `hyperframes` skill for composition rules and the `gsap` skill for timeline animation. Build static layout first, then add GSAP entrances/exits. Keep media, audio, captions, timing, aspect ratio, and output target explicit.
