# Compose mode

Create and iterate HyperFrames videos within the AOC asset/campaign system.

Use the `aoc-hyperframes` skill to confirm whether work belongs in reusable components, brand/system compositions, `_playgrounds/`, or a campaign pack. Use `aoc-hyperframes campaign create` for new campaign packs and `aoc-hyperframes workbench set` to focus preview. Use the `hyperframes` skill for composition rules and the `gsap` skill for timeline animation. Build static layout first, then add GSAP entrances/exits. Keep media, audio, captions, timing, aspect ratio, and output target explicit. Update `docs/composition-catalog.md`, matching shotlists, and asset inventory; run `aoc-hyperframes check` before handoff.
