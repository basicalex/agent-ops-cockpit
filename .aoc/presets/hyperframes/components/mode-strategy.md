# Strategy mode

Define or critique the brand strategy before creative production.

Work only inside the operator-approved brand frame. Read `DESIGN.md`, `hyperframes/docs/DESIGN.md`, and `hyperframes/docs/brand-strategy.md` when present. Capture audience, brand soul, promise/proof, voice, visual world, constraints, and off-brand boundaries. If the brand soul is vague, sharpen it instead of generating concepts. Output should update or reference `hyperframes/docs/brand-strategy.md`; do not generate image prompts or campaign compositions in this mode.
