## Commands
- `aoc-mem read`
- `aoc-mem search "<term>"`
- `aoc-mem add "<decision>"`

## Recording guidelines
- Capture the "why" behind decisions.
- Keep entries short and scoped.
- Record one decision per line.

## Guardrail
- Never edit `.aoc/memory.md` directly.
