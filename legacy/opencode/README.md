# Legacy OpenCode / OmO Artifacts

AOC is Pi-first. This directory contains archived OpenCode/Oh-My-OpenCode compatibility assets kept for historical regression, legacy imports, and reference only.

Active AOC runtime surfaces should not depend on these files.

Contents:

- `config/` — archived OmO/OpenCode policy templates.
- `scripts/` — archived OmO/OpenCode install, verify, and regression scripts.

Use only during an explicit legacy maintenance pass. Prefer Pi-native `.pi/**`, `bin/aoc-agent*`, and current AOC/Mind flows for normal operation.
