# Support

Thanks for using AOC. Here is how to get help and report issues.

## Quick Help

- Start with the docs in `README.md` and `docs/installation.md`.
- Run `aoc-doctor` to validate local dependencies.

## Issues

Use GitHub Issues for bugs and feature requests:
https://github.com/basicalex/agent-ops-cockpit/issues

Please include reproduction steps, your OS, terminal, and versions of Zellij
and Yazi.

## Discussions

If Discussions are enabled, use them for questions and ideas:
https://github.com/basicalex/agent-ops-cockpit/discussions

## Security

For security issues, please follow `SECURITY.md` and avoid public reports.
